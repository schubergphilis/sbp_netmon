sbp_netmon Cookbook
===================
This cookbook will install Microsoft Network Monitor 3.4 on a Windows system


Requirements
------------
The cookbook depends on the windows cookbook


Attributes
----------
default['netmon']['package_name']
default['netmon']['url']


Usage
-----
Just include `sbp_netmon` in your node's `run_list`
